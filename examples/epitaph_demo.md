# 墓志铭示例

## 墓志铭报告

### 📊 你的数字墓碑

- 发过的动态：2340 条
- 收获的赞：8700 个
- 最长沉默：47 天
- 最被记住的一句话："算了，先这样吧"

### 正经版

```
这里安息着一个认真活过的普通人。
她 25 岁时一个人去了一座陌生的城市，
没有带多少行李，但带了全部的勇气。
她不常说「我很在乎你」，
但每一次沉默，都是在用力忍着不说。
```

### 毒舌版

```
这里躺着一个发了 2340 条动态的人。
她最常说的话是「算了，先这样吧」。
她花了 30 年学会了扛事，
但一直没学会让别人帮她扛。
猫比她活得自在。
```

### 文艺版

```
她把勇气用在了出发上，
把沉默留给了在乎的人。
那些没说出口的话，
比她发过的 2340 条动态都重。
```

### 打工人版

```
工龄 8 年，加班 2000+ 小时。
最长连续沉默 47 天——不是因为没话说，
是因为太累了，连发朋友圈的力气都没有。
她想被记住的方式是「认真活过」，
但她大部分时间都在认真加班。
```

### 哲思版

```
她一直在等一个「准备好了」的时刻：
准备好了再写书，准备好了再学潜水，
准备好了再对那个人说「我很在乎你」。
她没有等到。
不是因为时间不够——
是因为「准备好了」这个时刻，从来不会自己到来。
```

### 🔮 灵魂追问

你想被记住的方式是"认真活过"——你上一次不是在"认真加班"而是在"认真活"，是什么时候？

你最想对谁说"我很在乎你"？为什么到现在还没说？

那本想写的书、想学的潜水——你在等什么"准备好了"的信号？

### ✍️ 如果从今天开始重写结局

今天就能做的一件事：给那个你一直想联系但没联系的人，发一条消息。不需要说"我很在乎你"——先说"最近怎么样"就够了。
