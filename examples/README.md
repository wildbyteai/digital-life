# 示例输出

这个目录里的文件全部是**虚构、脱敏、不可回溯到真实个人**的演示样例。

用途：

- 让 GitHub 访问者快速理解 skill 的实际产出长什么样
- 给后续 prompt 调整提供一个稳定的对照样本
- 避免把真实用户的数字痕迹提交到仓库

当前包含六组样例，覆盖全部 skill；另外包含一个 portable Life Skill Package demo：

- `past_life_demo.json` / `past_life_demo.md` — 👻 前世
- `cringe_archaeology_demo.json` / `cringe_archaeology_demo.md` — 💀 社死考古
- `ai_clone_demo.json` / `ai_clone_demo.md` — 🤖 AI替身
- `legacy_audit_demo.json` / `legacy_audit_demo.md` — 🪦 遗产清算
- `epitaph_demo.json` / `epitaph_demo.md` — 🪦 墓志铭
- `distilled_life_demo.json` / `distilled_life_demo.md` — 🧪 蒸馏人生
- `skill_packages/writing_style_demo/` — 🧪 Life Skill Package（虚构写作风格技能包，含 manifest、source hash、export 和 static tests）
- `skill_packages/decision_principles_demo/` — 🧪 Generated Life Skill Package（由虚构 `distilled_life_demo.json` 生成）
