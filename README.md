## 为什么做这个项目

我们每天在网上留下痕迹。

一条朋友圈，一次深夜搜索，一个凌晨三点的收藏，一次转发又秒删。

这些痕迹散落在 7 个平台、14 年时间里，我们从来不回头看。

不是因为忘了。**是因为不敢。**

苏格拉底说："未经审视的人生不值得过。"但在数字时代，审视自己的人生变成了不可能的任务——数据太多、平台太多、时间线太碎。

**Skill Box 做的事情很简单：帮你把碎片拼成一面镜子。**

不是为了好看，是为了看清。

## 🛠 内置工具

### 🪦 1. 遗产清算.skill

**哲学维度：海德格尔 — 向死而生（Sein zum Tode）**

> "人是向死存在者。只有直面死亡，才能本真地活着。"
> —— 海德格尔，《存在与时间》

你在互联网上留下了什么？如果明天你消失了，你的朋友圈会被谁翻出来？你的微博会在多久之后被系统回收？

**遗产清算帮你做一件事：以"死者"的视角回看自己的数字生命。**

```
📊 你的数字遗产报告
━━━━━━━━━━━━━━━━━━
📱 微信朋友圈：1,247 条
   最早一条：2013-06-15 "今天天气真好"
   被截图转发次数：3
   收到"哈哈哈哈"最多的：你发的那条摔倒视频

🐦 微博：892 条
   删除比例：67%（心虚了？）
   最后悔的一条："2016年一定要脱单！"

💀 QQ空间：2,341 条
   最多的一天：2012-08-20，发了 14 条
   转载的非主流日志：47 篇

⏳ 你总共在网上浪费了约 2,847 小时
   相当于：118 天 / 4 个月
   这些时间，够你写完一本书。
━━━━━━━━━━━━━━━━━━
🪞 最后的追问：
   如果你的所有数字痕迹明天就会消失，
   你最想留下的是哪一条？
```

**核心问题：** 你在数字世界里留下的，是遗产还是垃圾？

### 💀 2. 社死考古.skill

**哲学维度：汉娜·阿伦特 — 公共领域与私人领域**

> "在公共领域中，人通过言行显现自己；在私人领域中，人获得庇护。"
> —— 汉娜·阿伦特，《人的境况》

你十年前在 QQ 空间写的那些东西，当时是"公共表达"——你主动发的，主动展示的。但十年后，它们变成了你最想藏起来的私人尴尬。

**社死考古帮你看到一件事：你以为已经消失的自我，其实还在互联网的某个角落里活着。**

```
⏰ 社死时间线
━━━━━━━━━━━━━━━━━━
2009-03-14  QQ空间
  "今天的心情：💔 莪卜知道该说什么..."
  📎 配图：45度角仰望天空的自拍（已打码）

2011-07-22  人人网
  "分享日志：《真正爱你的10个表现》"
  👍 被3个人赞（都是你小号）

2013-11-08  百度贴吧
  "【求助】喜欢的女生不理我怎么办"
  💬 回复：兄弟别想了（73人赞同）

━━━━━━━━━━━━━━━━━━
🪞 最后的追问：
   那个用火星文写日志的少年/少女，
   真的比现在的你更差吗？
   还是说，他/她只是还没学会"体面"？
```

**核心问题：** 你删掉的那些"社死"内容里，是不是恰恰藏着你最真实的表达？

### 🤖 3. AI替身.skill

**哲学维度：图灵测试 — 什么是"你"？**

> "机器能思考吗？不如问：机器能模仿思考吗？"
> —— 艾伦·图灵，《计算机器与智能》

如果一个 AI 能完美模仿你说话，回复你老婆的消息不会穿帮，那它是不是"你"？

**AI替身逼你直面一个不舒服的问题：你的社交人格，到底有多少是"真正的你"？**

```
🤖 AI替身已上线
━━━━━━━━━━━━━━━━━━
学习数据：
  微信聊天记录：48,291 条
  朋友圈文案：1,247 条

人格特征：
  回复速度：偏慢（平均 3.2 分钟）
  常用词："哈哈" "好的" "随便" "没事"
  被动指数：92%（几乎从不主动找人）

关键发现：
  你在朋友圈的语气 vs 你在私聊的语气，相似度只有 37%。
  你有两个"你"：一个给别人看，一个给自己看。

  老婆："晚上吃什么？"
  你（AI替身）："你定，我都行"
  ✅ 通过 —— 和你本人回复一模一样

━━━━━━━━━━━━━━━━━━
🪞 最后的追问：
   如果 AI 替身可以完美替代你的社交互动，
   那"真正的你"在哪里？
   你上一次用"真实的语气"和人说话，是什么时候？
```

**核心问题：** 当你的社交行为可以被完全复制，"你"还剩下什么是不可替代的？

### 👻 4. 前世.skill

**哲学维度：佛教轮回 — 业力与因果**

> "一切众生，轮转五道，经百千劫，于多生中互为父母。"
> —— 《大般涅槃经》

前世不是玄学。它是一种**叙事方法**——帮你在看似随机的行为模式中，找到一条隐秘的因果线。

**前世帮你看到：你现在的习惯、恐惧、执着，可能比你以为的更古老。**

```
🔮 你的前世报告
━━━━━━━━━━━━━━━━━━
推演结果：
  🏛 你的前世：民国时期的书店老板

  证据链：
  - 你这辈子也爱囤书（买了200本没看）
  - 你害怕社交但喜欢观察人（书店老板标配）
  - 你总在深夜最清醒（前世守着孤灯校稿）
  - 你从不主动追人（前世也是等客人上门）

  因果分析：
  上辈子你看了太多人的故事，
  轮到自己时反而不知道怎么写了。
  所以这辈子你总是"想太多做太少"——
  这是前世留下的惯性。

━━━━━━━━━━━━━━━━━━
🪞 最后的追问：
   如果你相信前世塑造了今生，
   那你今天的行为，正在塑造谁的前世？
```

**核心问题：** 你以为是"性格"的东西，有多少只是"惯性"？

## 设计哲学

### "考古"而非"分析"

分析是冰冷的。考古是尊重时间的。

你十年前在 QQ 空间写的"莪の丗堺伱卜懂"不是错误数据，是你某个时刻的真实。我们不应该用"黑历史"三个字抹杀过去的自己。

### "镜子"而非"报告"

报告告诉你数字。镜子让你看见自己。数字可以造假，镜子不能。

### "向死而生"而非"活得更好"

这些 skill 帮你预演死亡——不是为了恐惧，是为了重新理解活着这件事值多少钱。
