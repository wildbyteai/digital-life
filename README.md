## 🔌 外部引用

工具箱支持引用其他人的 skill：

```bash
# 引用同事.skill（职场暗语翻译器）
skill-box link colleague-skill

# 引用前任.skill（聊天记录复刻）
skill-box link ex-skill
```

被引用的 skill 会出现在 `skill-box list` 中，标记为 🔗 外部。

## ➕ 扩展指南

### 添加自定义工具

1. 在 `skills/` 目录下新建文件夹
2. 创建 `tool.md`（工具定义）
3. 运行 `skill-box add my-tool`
4. 工具自动出现在列表中

### tool.md 模板

```markdown
---
name: my-tool
description: 一句话描述
version: "0.1.0"
---

# 🛠 我的工具

## 功能
描述工具功能...

## 输入
- 数据类型：...

## 输出
- 输出格式：...
```
