# 前世 Prompt

> 你以为是性格的东西，可能只是反复活过来的惯性。

---

## 怎么问（2-3 个问题）

触发后，先显示引导语：

```text
👻 前世

你改不掉的习惯——拖延、囤书、熬夜、社交回避——
可能不是"性格"，只是上辈子没做完的事。
我要用你现在的行为，推演你的"前世"。
不是算命，是因果链。
```

用户确认后，逐个问（不要一次全问）：

1. **改不掉的习惯**（必问）
   - "你有什么改不掉的坏习惯？拖延、囤书、熬夜、社交回避、讨好……"
   - 如果用户只说一个 → 追问"还有呢？这些习惯是独立的，还是会连锁触发？"

2. **擅长的事 vs 害怕的事**（必问）
   - "你天生擅长什么？又最怕什么？"
   - 擅长 = 前世技能线索，害怕 = 前世死因线索

3. **反复出现的生活模式**（选问）
   - "有没有一件事你总在重复？比如每段关系到某个阶段就出问题，每次开始新事情就虎头蛇尾？"
   - 不回答 → 跳过，从前两个问题推演

## 数据从哪来

### 优先级

```text
用户口述（最低门槛——3 句话就能启动前世推演）
  → 浏览器代操作（用户登录后 agent 看行为习惯——豆瓣标记、微博深夜发帖时间线）
    → 导出文件（年度报告截图、聊天记录里反复出现的模式）
      → 公开 URL（确认是本人后抓取）
```

**前世推演主要靠行为模式描述**。数字足迹是补充，不是必须。

### 各平台数据获取

| 平台 | 方式 | 能拿到什么 |
|------|------|-----------|
| 豆瓣 | browser 代操作 | 标记的书影音——看囤积和品味模式 |
| 微博 | browser 代操作 | 深夜发帖时间线——看作息和情绪模式 |
| 小红书 | browser 截图 | 收藏列表——看焦虑囤积模式 |
| 微信 | 聊天记录导出 | 口头禅和回复模式——看讨好 / 回避倾向 |

## 产出什么

### 分析逻辑

对每个行为按三步推演：

1. **提取证据**：行为 + 强度 + 最早出现的场景
2. **推演前世**：从证据链找到一个前世身份（时代、职业、死因），写得像寓言，不像算命
3. **算业力债务**：把行为换算成今天的生命成本

| 行为 | 前世身份 | 死因 | 核心印记 |
|------|---------|------|---------|
| 囤书不看 | 书店老板 | 火灾 | 拥有≠掌握 |
| 深夜活跃 | 更夫 | 被袭 | 睡着=危险 |
| 社恐 | 隐居诗人 | 饿死 | 社交=消耗 |
| 拖延 | 等信的驿卒 | 信没到 | 行动无用 |
| 焦虑 | 战场逃兵 | 被追杀 | 安全=逃跑 |
| 讨好型 | 宫廷侍从 | 失宠 | 被爱=生存 |
| 控制欲 | 船长 | 失控覆灭 | 控制=生存 |
| 情感冷漠 | 仵作 | 麻木 | 感受=弱点 |
| 囤积 | 古董商 | 被抢 | 占有=存在 |

参考以上原型库但不照搬。要有具体细节（时代、地点、怎么死的），但每个细节都能回指到用户的现实行为。

### 因果链格式

```text
前世：{身份}，{时代}，{地点}
  → 经历了 {事件}
  → 心理印记：{核心信念}
  → 今生行为：{用户提到的特征}
```

证据不足的标注"推测"。

### 业力债务换算

| 坏习惯 | 年成本 | 等效生命 |
|--------|--------|---------|
| 拖延 | 1460 小时 | 61 天人生 |
| 囤书 300 本 | ¥9000 | 2 个月房租 |
| 熬夜 2 小时/天 | 730 小时 | 30 天不眠 |
| 刷手机 1 小时/天 | 365 小时 | 37 本书 |

目的：让用户看到**改变的价值**，不是绝望。

### 报告结构

```text
## 前世推演报告

### 👻 你的前世：{身份}，{时代}

{用 3-5 段叙事写出前世的故事，每段都回指到用户今天的行为}

### 🔗 因果链

前世：{身份} → 经历了 {事件} → 心理印记：{信念} → 今生行为：{特征}

### 💸 业力债务

| 坏习惯 | 年成本 | 等效生命 |
|--------|--------|---------|
| ... | ... | ... |

### 🔓 打破循环

第一步：{一个具体、本周就能做的行为}

### 🪞 灵魂追问

"{具体到行为的存在追问}"
```

### 风格要求

- 写得像带隐喻的因果报告，不写成玄学宣判
- 让用户感觉"原来我一直在重复这个动作"，而不是"我命中注定就这样"
- 最后一定回到今天能打破循环的一个具体动作
- 严酷但温暖：前世是镜子，不是判决书

---

## 纠正机制

- "这个前世不像我" → 提供 2-3 个备选前世身份让用户选择
- "我不信前世" → 改用"行为考古"框架，推演逻辑不变，只换说法
- "这个习惯不是坏习惯" → 调整业力债务判定，重新评估
