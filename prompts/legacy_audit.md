# 遗产清算 Prompt：你在数字世界花了多少条命

## 任务

扫描用户的数字足迹，把抽象的"上网行为"换算成具体的生命单位——时间、连接、影响力。不是财务审计，是存在论审计：你留下了什么，又浪费了什么。

---

## 怎么问（Step 1：确认范围）

用户触发遗产清算后，问：

1. **想清算哪些平台？**
   - 微信朋友圈 / 微博 / 小红书 / 知乎 / 抖音 / B站 / 贴吧
   - 也可以跳过，说"所有能找到的都算"

2. **关注什么维度？**
   - "我花了多少时间"（时间审计）
   - "我删过多少东西"（后悔率）
   - "哪些内容值得被记住"（遗产盘点）
   - 也可以跳过，说"全部"

问完两个问题就可以开始。

---

## 数据从哪来（Step 2：获取数字足迹）

### 优先级

```
导出文件（最全——各平台数据导出 JSON/XML）
  → 浏览器代操作（用户登录后 agent 翻个人主页，统计发帖/点赞/粉丝数）
    → 用户口述（"我发了大概几百条朋友圈"）
      → 截图（平台年度报告、使用时长截图）
```

### 各平台获取方式

| 平台 | 方式 | 能拿到什么 |
|------|------|-----------|
| 微信朋友圈 | browser 代操作或导出 | 发帖数量、时间跨度、互动数据 |
| 微博 | browser 代操作 | 发帖数、转发数、粉丝数、注册时间 |
| 小红书 | browser 截图 | 笔记数、收藏数 |
| 知乎 | web_fetch 或 browser | 回答数、赞同数、关注话题 |
| 抖音 | 用户截图 | 使用时长、点赞数 |
| B站 | web_fetch 或 browser | 投币/收藏/观看历史 |

### 关键数据点

每个平台至少需要：
- 注册/开始使用的大致时间
- 总发帖/内容数量
- 已删除内容数量（如果能拿到）
- 最高互动的一条内容

---

## 产出什么（Step 3：数字遗产报告）

### 换算公式

| 原始数据 | 换算 |
|---------|------|
| 刷手机小时数 | ÷24 = 天数，÷365 = 年数 |
| 发帖数量 | ×平均阅读量 = 影响人次 |
| 删除比例 | 已删 / 总发 = 后悔率 |
| 沉默天数 | ×日均互动 = 错过的连接数 |

### 输出报告结构

```
## 数字遗产报告：{用户昵称}

### ⏱️ life_in_numbers

你在数字世界花的时间，换算成生命单位：
- 刷手机总计：{X}小时 = {X}天 ≈ {X}年生命
- 可以换：{N}本书 / {N}次旅行 / {N}个技能
- 最高沉默纪录：{X}天（期间错过了什么？）

### 📱 各平台盘点

| 平台 | 活跃时间 | 发帖数 | 删除率 | 最有价值内容 |
|------|---------|--------|--------|------------|
| 微信 | 8年 | ~1200条 | 15% | [具体那条] |
| ... | ... | ... | ... | ... |

### 😬 后悔模式

识别三类后悔：
1. 即时后悔（发完就删）：{N}次
2. 累积后悔（多年后开始后悔整个时期的风格）：{描述}
3. 社会性后悔（被特定人看到后的后悔）：{描述}

### 🔮 存在论三问

1. 你花在数字世界的时间，如果用来做一件事，你会做什么？
2. 你在网上建立的连接，有多少是真实的？
3. 如果明天所有平台关闭，你最遗憾失去的是什么？

### 🪞 灵魂追问

具体到行为的问题（不要鸡汤）：
示例："你花在刷手机上的时间，够读完几本买了没看的书？"
```

### 风格要求

- 数据要具体、可感知——不说"你浪费了很多时间"，说"你花在刷手机上的 2847 小时，够读完 28 本书"
- 不说教——只呈现数据，让用户自己感受
- 结尾追问必须是用户能立刻回忆起来的具体行为
