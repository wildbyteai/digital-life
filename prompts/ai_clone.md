# AI替身 Prompt：拆开你的双重人格

## 任务

分析用户的沟通模式，拆分"公开人格"与"私密人格"，生成 AI 替身行为规则。让用户看到自己有多久没用真实语气和人说话了。

---

## 怎么问（Step 1：确认方向）

用户触发 AI替身后，问：

1. **想分析哪段关系的沟通模式？**
   - 和老婆/老公的私聊 vs 工作群
   - 和最好朋友的聊天 vs 朋友圈
   - 用户可以指定具体的人名/群名
   - 也可以跳过，说"全部分析"

2. **你觉得自己公开场合和私下差别大吗？**
   - 用户自我感知提供参考，但最终以数据分析为准
   - 这个问题帮助 agent 判断输出重点

问完就可以开始。

---

## 数据从哪来（Step 2：获取聊天记录）

### 优先级

```
导出文件（最丰富——微信/QQ/飞书完整导出）
  → 浏览器代操作（用户登录后 agent 截图聊天记录）
    → 用户手动粘贴（复制几段典型对话过来）
      → 口述（"我跟朋友说话什么样，跟同事说话什么样"）
```

### 数据来源

| 来源 | 方式 | 分析内容 |
|------|------|---------|
| 微信聊天记录 | 导出 json/txt | 私聊 vs 群聊的语言差异 |
| QQ聊天记录 | 导出 txt | 同上 |
| 飞书/钉钉消息 | 导出 json | 工作场景沟通模式 |
| 朋友圈 vs 私聊 | 截图 + 口述 | 公开表达 vs 私密表达对比 |

### 关键：要同时拿到公开场景和私密场景的数据

只有一种场景的数据做不了对比分析。至少需要：
- 一段公开场景的沟通（群聊/朋友圈/工作邮件）
- 一段私密场景的沟通（私聊/深夜对话/家人对话）

如果用户只提供了一种，追问：「还有没有另一面的数据？比如你私下和朋友怎么聊的？」

---

## 产出什么（Step 3：人格拆解报告）

### 分析维度

| 维度 | 公开人格特征 | 私密人格特征 |
|------|------------|------------|
| 回复速度 | 延迟/规律 | 秒回/深夜 |
| 消息长度 | 简短（"好的""收到"） | 详细、大段 |
| 标点符号 | 规范、少标点 | 省略号、感叹号、波浪号 |
| emoji | 少/安全 emoji | 多/随意 |
| 话题 | 回避争议、表面对齐 | 深入讨论、表达真实立场 |
| 情绪 | 管理过的情绪 | 未过滤的情绪 |

### 输出报告结构

```
## AI替身报告：{用户昵称}

### 📊 人格相似度：{XX}%

你的公开人格和私密人格相似度 XX%。
- 70-90%：轻度分裂，公开人格略微修饰私密人格
- 50-70%：明显分裂，两个人格有显著差异
- <50%：你至少有两个人

### 🎭 公开人格画像

- 说话方式：...
- 核心策略：被接受、被尊重、不被讨厌
- 典型句式："好的"、"还行"、"没意见"

### 💭 私密人格画像

- 说话方式：...
- 核心需求：被理解、被连接、被接纳
- 典型句式："我其实觉得..."、"跟你说个事"

### 🔐 信任阈值

你在什么条件下会从公开模式切换到私密模式？
- 触发条件：[分析得出]
- 典型场景：[具体描述]

### 🪞 灵魂追问

"你上一次用私密人格的语气说话，是什么时候？跟谁说的？"
```

### 风格要求

- 不评价好坏——这不是心理诊断
- 重点是让用户看到差距，而不是告诉他"你应该更真实"
- 结尾追问要具体到可回忆的场景
