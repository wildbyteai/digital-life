# 蒸馏人生方法论 / Distilled Life Reference

## 核心定义

蒸馏人生是把一个人的经历、决策、表达、边界和纠正记录，转成可复用 Life Skill Profile 的方法。

它不是数字永生，也不是人格复制。它更像一个可版本管理的自我模型：

```text
原始痕迹 → 证据 → 解释 → 技能资产 → 评测问题 → 纠正 → 新版本
```

## 关键对象

### Life Story

一个具体经历片段。

字段：

- context：发生在什么场景
- stakes：当时的代价和压力
- options：有哪些选择
- actions：做了什么（列表）
- reasoning：为什么这么做（列表）
- outcome：结果如何
- lesson：后来怎么理解
- quotes：能代表当时状态的原话（列表）
- permission：能否被使用

### Skill Asset

从 Life Story 中提炼出的可复用能力。

字段：

- id：技能资产 ID
- name：技能名称
- use_when：什么时候适用（列表）
- method：怎么做（列表）
- heuristics：判断口诀或启发式（可选）
- examples：例子（可选）
- counterexamples：反例（列表）
- failure_modes：误用风险（可选）
- evidence_ids：证据来源（可选）
- eval_prompts：测试问题（列表）

### Decision Model

这个人通常怎么判断。

不要只写“理性/感性”。要写：

- 他先看什么
- 他怕什么代价
- 他会问谁
- 他会怎么试错
- 哪些东西会让他改变想法
- 哪些事情他会拖延或回避

### Expression Model

这个人通常怎么表达。

包括：

- 句子长短
- 是否喜欢列表
- 常用口头禅
- 拒绝方式
- 对熟人/陌生人的差异
- 哪些语气不像他

### Boundary Rule

定义哪些事不能让 AI 代替用户做。

重点：

- 对外发言
- 承诺合作
- 金钱、职业、关系重大决定
- 私人信息披露
- 原始聊天记录引用

### Evidence Trace

每个重要判断都应回答：

- 依据来自哪里？
- 是原话、行为、用户复述，还是推测？
- 证据新鲜吗？
- confidence 是 high、medium 还是 low？
- 能给哪个 audience 看？

## 蒸馏流程

### Step 1：收集材料

优先收集高信号材料：

- 一次真实决策
- 一段真实表达
- 一个冲突/妥协案例
- 一个失败复盘
- 一个别人容易误解用户的地方
- 一个用户明确说“我不会这么说/不会这么做”的反例

### Step 2：拆成证据

把材料拆成：

- facts
- quotes
- choices
- tradeoffs
- outcomes
- emotions
- constraints
- corrections

### Step 3：提炼模式

问：

- 这是一次偶然，还是重复出现？
- 这件事体现的是偏好、能力、恐惧，还是边界？
- 有没有反例？
- 这个结论能不能被证据支撑？

### Step 4：生成技能资产

一个好 Skill Asset 应该能被别人或未来的自己使用。

坏例子：

> 他很努力。

好例子：

> 面对新技术时，他会先看综述，再跑一个最小 demo，通过动手建立判断；但如果只是短期交付，不应该过度自研。

### Step 5：生成评测问题

每个 Skill Asset 至少生成 3 个问题：

1. 直接应用问题
2. 边界/反例问题
3. 表达风格问题

例如：

- “什么时候应该先做 demo？”
- “什么时候不该自研？”
- “用他的语气解释这件事。”

### Step 6：纠正和版本管理

用户纠正后，不要只改文字。要记录：

- 哪个判断被推翻
- 哪条证据被重新解释
- 哪个 skill asset 需要更新
- confidence 如何变化

## 质量标准

一份好的蒸馏报告应该满足：

- 具体：有真实场景，不是抽象人格标签
- 可追溯：能看到 evidence
- 可复用：能转成方法，而不是只讲故事
- 有边界：知道什么时候不能用
- 可纠正：用户指出不对后能更新
- 不冒充：不替用户做行动或承诺

## 与其他 5 个 skill 的关系

- 前世：看惯性
- 社死考古：看表演和羞耻
- AI替身：看公开人格和私人人格
- 遗产清算：看数字痕迹和时间
- 墓志铭：看终点和未完成
- 蒸馏人生：把经历转成可复用判断和表达能力

一句话：前 5 个 skill 帮用户照见自己，蒸馏人生帮用户复用自己。
