# 🔧 Skill Box — 数字人格考古工具箱

> 你发了 3 万条朋友圈，收获 2 个真心朋友。
> 你花了 4 个月的寿命在网上冲浪。
> 你以为你在记录生活，其实生活在记录你。

## 这不是工具箱，是一组镜子

> "未经审视的人生不值得过。" —— 苏格拉底

我们每天在网上留下痕迹：一条朋友圈、一次深夜搜索、一个凌晨三点的收藏。这些痕迹散落在 7 个平台、14 年时间里，我们从来不回头看。

不是因为忘了，是因为不敢。

Skill Box 做的事情很简单：**帮你把碎片拼成一面镜子。**

不是为了好看，是为了看清。

## 使用方式

```bash

# 列出所有工具

# 运行某个工具

skill-box run cringe-archaeology # 社死考古
skill-box run ai-clone           # AI替身

# 添加自定义工具
skill-box add my-custom-tool

# 引用外部 skill
skill-box link colleague-skill
skill-box link ex-skill
```

## 设计哲学

### 为什么是"考古"而不是"分析"？

考古意味着**尊重时间的沉淀**。你十年前在 QQ 空间写的"莪の丗堺伱卜懂"不是错误数据，是你某个时刻的真实。我们不应该用"黑历史"三个字抹杀过去的自己。

### 为什么是"镜子"而不是"报告"？

报告告诉你数字。镜子让你看见自己。数字可以造假，镜子不能。

### 为什么要"向死而生"？

海德格尔说，人只有直面死亡，才能真正活着。这些 skill 帮你预演死亡——不是为了恐惧，是为了**重新理解活着这件事值多少钱**。
