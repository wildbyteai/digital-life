# 前世.skill

> 根据你的性格测试、行为数据、童年照片，AI 推演你的"前世"是什么。

## 哲学锚点

**佛教轮回 — 业力与因果**

> "一切众生，轮转五道，经百千劫，于多生中互为父母。"
> —— 《大般涅槃经》

前世不是玄学，它是一种**叙事方法**。

你的行为模式不是随机的。你囤书不看，你深夜最清醒，你从不主动追人，你容易焦虑——这些不是"性格"，是**惯性**。

前世帮你做一件事：在看似随机的行为中，找到一条隐秘的因果线。让你看到：你以为是"我就是这样的人"的东西，可能只是一种古老的习惯。

**如果前世塑造了今生，那今生正在塑造谁的前世？**

## 功能

用你的数字痕迹反推你的"前世"——不是算命，是行为模式的因果叙事。

## 输入

| 数据 | 格式 | 说明 |
|------|------|------|
| 性格测试 | json/text | MBTI、大五人格等 |
| 行为数据 | json | 作息、消费、社交模式 |
| 童年照片描述 | text | 你描述或 AI 分析 |
| 数字足迹 | json | 发言风格、兴趣偏好 |

## 执行流程

1. 收集性格和行为数据
2. 匹配历史人物/职业原型数据库
3. 构建"前世因果链"（行为→前世对应→行为模式命名）
4. 计算"业力债务"——这辈子还没还清的前世债
5. 生成前世报告 + 存在追问

## 核心追问

- 如果你的"性格"只是一种"惯性"，那什么是"真正的你"？
- 如果前世塑造了今生的行为模式，那**今生**有没有可能打破这个循环？
- 你今天的行为，正在塑造谁的前世？
