# 前世.skill

> 根据你的性格测试、行为数据、童年照片，AI 推演你的"前世"是什么。

## 功能

用你的数字痕迹反推你的"前世"——不是玄学，是行为模式的因果联想。

## 输入

| 数据 | 格式 | 说明 |
|------|------|------|
| 性格测试 | json/text | MBTI、大五人格等 |
| 行为数据 | json | 作息、消费、社交模式 |
| 童年照片描述 | text | 你描述或 AI 分析 |
| 数字足迹 | json | 发言风格、兴趣偏好 |

## 执行流程

1. 收集性格和行为数据
2. 匹配历史人物/职业原型数据库
3. 构建"前世因果链"（行为→前世对应→解释）
4. 生成带幽默感的前世报告
5. 输出"解法"（其实没什么用，但看着有道理）

## 隐私

- 全部本地分析
- 仅供娱乐，不作为任何决策依据
- "前世"纯属虚构，如有雷同纯属巧合
