# AI替身.skill

> 训练一个你的"社交替身"，自动帮你回消息。

## 哲学锚点

**图灵测试 — 什么是"你"？**

> "我们只能看到自己的影子。"
> —— 柏拉图，《理想国》洞穴寓言

如果一个 AI 能完美模仿你说话——你老婆发现不了，你朋友发现不了，甚至你妈都发现不了——那它是不是"你"？

这不只是一个技术问题。这是关于**身份本质**的哲学追问。

你每天在微信上说"哈哈"、"好的"、"随便"、"没事"——这些是"你"吗？还是只是"你在社交中的投影"？

**AI替身逼你看到：你的社交人格，有多少是"真正的你"，有多少是"你为了让别人舒服而表演的你"。**

## 功能

从你的聊天记录中学习说话方式，生成一个能替你回消息的 AI。然后问你一个不舒服的问题：**如果它能替代你，那"你"还剩什么？**

## 输入

| 数据 | 格式 | 说明 |
|------|------|------|
| 微信聊天记录 | txt/csv | 和不同人的对话 |
| 朋友圈文案 | json | 你的公开发言 |
| 微博发言 | json | 更随意的表达 |
| 短信记录 | xml/csv | 正式/紧急场景 |

## 输出

```json
{
  "persona": {
    "public_self": {
      "tone": "温和、被动、避免冲突",
      "top_words": ["哈哈", "好的", "随便", "没事"],
      "initiate_ratio": 0.08
    },
    "private_self": {
      "tone": "直率、偶尔毒舌、深夜更真",
      "top_words": ["真的假的", "烦死了", "算了"],
      "initiate_ratio": 0.23
    },
    "self_similarity": 0.37,
    "insight": "你在朋友圈的语气 vs 你在私聊的语气，相似度只有37%。你有两个'你'。"
  },
  "clone": {
    "public_clone": { "test": "✅ 通过" },
    "private_clone": { "test": "⚠️ 偶尔穿帮" },
    "key_question": "如果两个替身都能替代你，那'真正的你'在哪里？"
  }
}
```

## 执行流程

1. 解析聊天记录，提取对话模式
2. 分析高频词、回复速度、emoji 使用
3. 拆分"公开人格" vs "私密人格"
4. 构建两个 clone
5. 对比两个版本的相似度
6. 输出存在追问

## 洞察层

### 你的"公开人格"发现了什么？

- 常用词：["哈哈", "好的", "随便", "没事"]
- 被动指数：92%
- 回复速度：偏慢（平均 3.2 分钟）
- 情绪波动：极低（你藏得很好）

### 你的"私密人格"发现了什么？

- 常用词：["真的假的", "烦死了", "算了"]
- 主动指数：23%
- 回复速度：快（平均 47 秒）
- 情绪波动：高（你只在信任的人面前放肆）

### 关键问题

- 你在朋友圈的语气 vs 你在私聊的语气，相似度只有 **37%**。
- 你有两个"你"：一个给别人看，一个给自己看。
- **你上一次用"真实的语气"和人说话，是什么时候？**

## ⚠️ 警告

- 对老板可以用（反正他也分不出来）
