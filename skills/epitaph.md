# 墓志铭.skill

> 根据你的人生轨迹自动生成墓志铭。

## 哲学锚点

**塞涅卡 — 论生命的短促**

> "我们的生命并不短，是我们浪费了太多。"
> —— 塞涅卡，《论生命之短促》

你不是活了三十年。你是**浪费了三十年中的一大部分**。剩下的那部分，才是真正活着的。

墓志铭帮你在活着的时候，读到自己死后别人会怎么写你。

这很残忍，但很必要。因为大多数人在写墓志铭之前，从来没认真想过一个问题：**你想被记住的方式，和你正在过的生活，一致吗？**

如果不一致——今天还来得及改。

## 功能

用你的数字足迹拼出你的人生叙事，刻在数字墓碑上。在活着的时候，看到自己的结局。

## 输入

| 数据 | 格式 | 说明 |
|------|------|------|
| 社交平台数据 | json | 发言、互动、点赞 |
| 消费记录 | json | 支付宝/微信年度账单 |
| 行为数据 | json | 作息、出行、习惯 |
| 人生经历 | text | 你主动输入的关键事件 |

## 执行流程

1. 统计数字足迹（发了多少、互动多少、沉默多久）
2. 提取人生叙事（高光时刻、低谷期、转折点）
3. 识别"遗憾模式"（想做没做的事）
4. 生成墓志铭（多风格版本）
5. 生成存在追问

## 墓志铭风格

### 正经版
```
此人来过，爱过，发过朋友圈。
最后一条消息是已读未回。
愿在另一个世界，有人秒回。
```

### 毒舌版
```
此人最大的成就是：
在35个群里保持了"免打扰"状态。
在朋友圈维持了"阳光积极"人设。
在支付宝贡献了4.7万给美团外卖。
安息吧。你的外卖到了。
```

### 文艺版
```
他活在屏幕里，
死在通知栏中。
最后一条消息是验证码，
没人帮他收。
```

### 打工人版
```
此人全年无休。
朋友圈永远在线，
工位永远有人。
除了他的工资单。
```

### 哲思版
```
塞涅卡说："我们的生命并不短，是我们浪费了太多。"
此人的生命：30年。
真正活着的时间：待计算。
浪费在屏幕上的时间：4个月。
浪费在犹豫上的时间：无法统计。
浪费在假装上的时间：一辈子。

他用一生证明了一件事：
人可以同时在线，又同时缺席。
```
