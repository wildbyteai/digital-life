# 墓志铭.skill

> 根据你的人生轨迹自动生成墓志铭。

## 功能

用你的数字足迹拼出你的人生叙事，刻在数字墓碑上。

## 输入

| 数据 | 格式 | 说明 |
|------|------|------|
| 社交平台数据 | json | 发言、互动、点赞 |
| 消费记录 | json | 支付宝/微信年度账单 |
| 行为数据 | json | 作息、出行、习惯 |
| 人生经历 | text | 你主动输入的关键事件 |

## 输出

```
╔════════════════════════════════════════════╗
║                                            ║
║                   R.I.P.                   ║

║         收获 2 个真心朋友                    ║

║                                            ║
║         最后一条朋友圈：                      ║
║         "今天也要加油鸭🦆"                   ║
║         获赞：3（含 1 个老妈）                ║
║                                            ║
║         他最大的遗憾是：                      ║
║         想去的地方都没去                      ║
║         想说的话都没说                        ║
║         想删的黑历史都没删                    ║

║         在最难的日子里没有放弃                 ║

║                                            ║

║                                            ║

```

## 执行流程

1. 统计数字足迹（发了多少、互动多少、沉默多久）
2. 提取人生叙事（高光时刻、低谷期、转折点）
3. 识别"遗憾模式"（想做没做的事）
4. 生成墓志铭（正经版 + 毒舌版）
5. 输出多种风格供选择

## 墓志铭风格

### 正经版
```
此人来过，爱过，发过朋友圈。
最后一条消息是已读未回。
愿在另一个世界，有人秒回。
```

### 毒舌版
```
此人最大的成就是：
在35个群里保持了"免打扰"状态。
在朋友圈维持了"阳光积极"人设。
在支付宝贡献了4.7万给美团外卖。
安息吧。你的外卖到了。
```

### 文艺版
```
他活在屏幕里，
死在通知栏中。
最后一条消息是验证码，
没人帮他收。
```

### 打工人版
```
此人全年无休。
朋友圈永远在线，
工位永远有人。
除了他的工资单。
```

## 隐私

- 全部本地分析
- 墓志铭仅供娱乐
- 死后不会真的刻上去（除非你真的想）
