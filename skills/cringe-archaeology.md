# 社死考古.skill

> 自动翻遍你的QQ空间、人人网、贴吧老帖，把十年前的中二发言全部挖出来。

## 哲学锚点

**汉娜·阿伦特 — 公共领域与私人领域**

> "在公共领域中，人通过言行显现自己；在私人领域中，人获得庇护。"
> —— 汉娜·阿伦特，《人的境况》

十年前你在 QQ 空间写的东西，当时是"公共表达"——你主动发的，主动展示给所有人看的。但十年后，它们变成了你最想藏起来的私人尴尬。

这中间发生了什么？

**不是你变"成熟"了，是你学会了"表演体面"。**

那个用火星文写日志的少年，那个在贴吧认真提问的少年，那个转发"真正爱你的10个表现"的少年——他们不是"黑历史"，他们是你的前传。

社死考古做的不是嘲笑，是**还原本真**。让你看到：在学会"体面"之前，你表达过更真实的自己。

## 功能

考古你的互联网黑历史，不是为了嘲笑，是为了找回"还没学会表演"的自己。

## 输入

| 平台 | 格式 | 说明 |
|------|------|------|
| QQ空间 | 导出 xml/html | 包含日志、说说、留言板 |
| 人人网 | 导出 html | 包含状态、日志、相册 |
| 百度贴吧 | 账号页面 | 帖子和回复 |
| 新浪博客 | 导出 xml | 博客文章 |
| 校内网 | 导出 html | 人人网前身 |

## 执行流程

1. 解析导出文件（XML/HTML/JSON）
2. 按时间排序所有内容
3. 识别火星文、非主流用语、emoji 滥用
4. 计算"社死指数"（0-100）
5. 计算"本真指数"——越不"体面"的内容，本真度越高
6. 生成时间线 + 对比报告

## 双重指标

| 指标 | 含义 | 反直觉发现 |
|------|------|------------|
| 社死指数 | 内容有多"尴尬" | 越高越好笑 |
| 本真指数 | 内容有多"真实" | 越"社死"越本真 |

## 关键洞察

社死指数和本真指数呈**反比关系**：

- 2009-2012：高社死 + 高本真（你还不知道什么叫"人设"）
- 2013-2015：社死下降 + 本真下降（你学会了"体面"）
- 2016-至今：低社死 + 低本真（你变成了一个"安全"的人）

**社死考古的真正价值：让你看到自己是从什么时候开始学会"不真实"的。**
